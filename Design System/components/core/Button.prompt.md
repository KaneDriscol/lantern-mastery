The primary interactive control — bronze-filled CTA, cream secondary, or on-dark secondary.

```jsx
<Button variant="primary">Learn Now</Button>
<Button variant="secondary">Explore Traditions</Button>
<Button variant="secondary-on-dark">View Sequence</Button>
```

Variants: `primary` (default, darkens to `--primary-active` on press), `secondary` (hairline border on cream), `secondary-on-dark` (never inverts to a light button on dark surfaces). `disabled` dims to 45% opacity.
