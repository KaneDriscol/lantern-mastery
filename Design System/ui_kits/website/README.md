# Lantern — Website UI Kit

Interactive recreation of the Lantern marketing/education website, composed from the design system's own components (no re-implementation of primitives).

Screens: Home (hero, feature grid, technique + source cards, principle callout, CTA band), Techniques & Traditions (search, category tabs, comparison grid, technique sequence, featured virtue), Practice (drill log, add-entry), Resources (connector tiles, source references, cookie consent).

Navigate via the top nav; "Learn Now" jumps to Techniques. All state is client-side/fake — this is a visual + interaction recreation, not production code.
